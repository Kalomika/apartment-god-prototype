# Top Down Visual Style

## Archived Direction

This document records the earlier top down prototype direction.

It is no longer the active playable visual target.

The active visual target is now the side view hard cam sprite oriented wrestling game described in:

`wrestling_sim/docs/hard_cam_sprite_direction.md`

## Archived Notes

The earlier test focused on:

```text
absolute top down camera
black and white line art
simple top down ring
simple overhead wrestler sprites
manager choice UI below the match
```

Those notes remain useful as historical exploration, but they should not override the current hard cam sprite direction.
