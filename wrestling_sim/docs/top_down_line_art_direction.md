# Grapple Gods Top Down Line Art Direction

## Archived Direction

This document records the earlier top down line art experiment.

It is no longer the active playable visual target.

The active visual target is now the side view hard cam sprite oriented wrestling game described in:

`wrestling_sim/docs/hard_cam_sprite_direction.md`

## Archived Notes

The earlier experiment was based on:

```text
faceless overhead head shapes
visible shoulders from above
rounded torso silhouettes
thin arms with elbows and wrists
staggered legs and small feet
simple gear shapes
thin gray or black outline
```

Those ideas should not drive the current playable view. The current playable view should prioritize a hard cam ring, side view sprite readability, and 8 fps authored animation cycles.
