# Grapple Gods Aesthetic Reference Library

This folder is the shared visual source of truth for Grapple Gods.

Use it for style targets, prompt anchors, sprite guides, rig notes, generated visual references, and editable vector model sheets that should stay consistent across future GPT passes.

## Current folders

- `top_down_wrestling/`, realistic overhead line art references for wrestlers, rings, rig parts, and first playable visual testing.

## Usage rule

Before creating or revising Grapple Gods visuals, check the relevant folder here first. Keep new visual reference files inside `wrestling_sim/assets/aesthetics/` so future GPTs do not confuse Grapple Gods with the unrelated apartment game systems.
