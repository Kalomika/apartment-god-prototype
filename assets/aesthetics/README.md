# Aesthetic Reference Library

This folder is the shared art direction home for GPT assisted work on the game.

Use it for style targets, prompt anchors, sprite guides, rig notes, and visual reference assets that should stay consistent across future passes.

## Current folders

- `wrestling_topdown/`, realistic top down line art reference for the wrestling prototype, clean vector style, adult wrestler proportions, readable overhead silhouettes, rig friendly separated body logic.

## Usage rule

Before creating or revising game visuals, check the relevant folder here first, then keep new assets and notes in the same folder family so every future GPT can find the same aesthetic source of truth.
