# Reusable Style Prompt, Wrestling Top Down

Use this as the prompt anchor when generating or revising visuals for the wrestling prototype.

```text
Create a realistic top down wrestling game sprite reference in clean vector style line art.

Use an overhead orthographic camera. The figure should read like a real adult wrestler seen from directly above, with broad shoulders, muscular torso, attached arms, visible hair crown, compact hips, and boots more visible than the full legs. Avoid cute proportions, chibi, mascot styling, toy shapes, thick comic inking, and side view posing.

The visible character should feel like one natural connected body. Internally, the rig can be separated into head, torso and hips core, upper arms, forearms and hands, upper legs, lower legs, and boots. Hide those separations through overlap, draw order, and clean silhouettes.

Line work should be crisp, thin to medium, clean, production friendly, and readable at game scale. Use simple flat color with very restrained detail. The anatomy should be simplified but believable, not painterly and not 3D rendered.

Create two first test wrestlers: one dark haired wrestler with dark gear, and one light haired wrestler with tan or gold gear. Keep their body shape similar for testing, but make their hair and gear clearly different.

Pose requirements: neutral idle standing, walk step, run step, crouched ready stance, grapple ready stance with arms raised and open, supine grounded pose, seated cross legged gimmick pose.

Output should work as a Phaser friendly sprite or rig reference, with isolated pieces or clear pose cells, lots of empty space, and transparent background when possible.
```

## Short version

```text
Realistic overhead vector line art wrestlers, adult proportions, broad shoulders, top down readable silhouettes, rig friendly body parts hidden by overlap, dark hair dark gear test wrestler, light hair gold gear test wrestler, no chibi, no mascot, no thick comic ink.
```
