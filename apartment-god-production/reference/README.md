# Apartment God Visual Reference Archive

Purpose: this folder stores visual reference material and study notes for Apartment God.

This folder is required for visual production discipline. It is not a runtime asset folder.

Reference files are study material. Runtime game assets belong in runtime asset paths, not here.

Use this archive to preserve:

```txt
1. Kam supplied screenshots.
2. Kam supplied website images.
3. Kam supplied game screenshots.
4. External visual targets linked by Kam.
5. Prior approved Apartment God screenshots.
6. Prior rejected Apartment God screenshots.
7. Top down anatomy studies.
8. Vehicle, furniture, clothing, pose, animation, and environment references.
```

Do not create visual systems from memory only.

For any important visual pass, inspect the development matrix first, then inspect this archive for relevant references or rejected examples.

Recommended structure:

```txt
approved-style-targets/
rejected-visuals/
top-down-anatomy/
vehicles/
characters/
furniture/
animation/
screenshots/
```

Rules:

```txt
1. Name files clearly.
2. Add notes explaining what the reference proves.
3. Mark rejected examples clearly so future agents do not repeat them.
4. Mark approved style targets clearly so future agents understand the quality bar.
5. Do not confuse reference material with final runtime assets.
6. Study the reference, break down why it works, then build an Apartment God version.
```
