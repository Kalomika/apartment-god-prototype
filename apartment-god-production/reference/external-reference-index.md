# Apartment God External Visual Reference Index

This index records external visual targets Kam wants Apartment God agents to study.

Reference entries do not automatically become runtime assets. They are used to study construction, camera angle, animation discipline, silhouette, finish level, and what quality bar Apartment God should aim toward.

---

## Character Creator 2D, Modern by SmallScaleInt

Source:

```txt
https://smallscaleint.itch.io/character-creator-2d-modern
```

Use this reference for:

```txt
1. Top down or angled top down 2D character quality.
2. Modular clothing and body part construction discipline.
3. Mature sprite proportions instead of toy or mascot proportions.
4. Controllable outline and silhouette clarity.
5. 8 direction animation discipline.
6. Adjustable animation frame rate and anime motion timing awareness.
7. Bikes and vehicles as systems that need real mounted poses and directional animation.
8. The level of polish expected before Apartment God should call a visual direction acceptable.
```

Do not use this reference to:

```txt
1. Make Apartment God isometric when the target is true top down.
2. Copy one to one without transforming into Apartment God specific design.
3. Justify symbolic procedural shortcuts.
4. Replace reference study with memory only guessing.
```

Study goal:

```txt
Start with the quality bar.
Break down why it works.
Understand the top down construction.
Build an Apartment God version.
Strip away direct copying.
Keep the quality.
```
